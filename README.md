### 🚀 VConVert v1.0.0

This is the first public release of VConvert — a fast, modern, musical‑themed YouTube converter for macOS and Windows.

#### Features
- MP3 (320kbps), WAV, M4A, MP4 (1080p)
- Modern Tkinter GUI with musical instrument theme
- Local processing (no servers)
- Safe, open‑source, cross‑platform
